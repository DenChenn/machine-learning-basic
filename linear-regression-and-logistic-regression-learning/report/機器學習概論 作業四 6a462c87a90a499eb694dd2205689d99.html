Linear Regression:

  • Data Generation:

先定義一個 get_y 函數，主要用途就是對於下面每一個利用 uniform distibution 隨機生成的x，都能夠透過這個函數得到相對應的y。

函數的實作方式是根據作業給的polynomial，並且epsilon是利用計算出來的平均值以及標準差透過np的 normal distribution 套件生成。

  • Data Preprocessing and Model Training:

首先我們將剛剛的x座標reshape過後送進去 PolynomialFeature 套件將每一個x轉換為feature vector，並且根據dimension的不同，我建構了四種不同的模型。

然後我們將x指定範圍內的點透過 linspace 均勻的切分成100個點，然後將這些點都轉成不同 dimension 的 feature vector，並且利用剛剛的四種對應dimension的模型分別做predict。

最後我們將剛剛對於不同dimension的(x, y)會製成圖表(y是model predict x 的結果)，加上利用一些套件將訓練出來的model的係數組裝成latex表示法，放在圖表右上。
