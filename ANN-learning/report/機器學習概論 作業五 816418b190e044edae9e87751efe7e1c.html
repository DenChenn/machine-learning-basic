Data Preprocessing

首先將資料打亂。

之後將剛剛很醜的dataframe的label以及feature分開來，並且對於所有feature進行one-hot encoding。

做完大概長這樣。

接下來再處理label的部分。我將每一個label利用正整數來做代表，因為我用的是pytorch所以我的input target 不可以是string形式，實作方式則是簡單的使用雙向map對應。

然後進行資料集的分割，按照需求是7:3。

並且將分割好的集合轉乘pytorch tensor，等等訓練模型跟驗證模型會用到。

Model Construction:

利用pytorch 簡單建立一個單層模型。隨機設定中間的dimension成500。

其實我嘗試過利用2層以及3層的模型，但是表現都不太好，最扣到只剩下一層後表現是最好的。

然後誤差函數的部分我使用CrossEntropyLoss的方法實作。

最佳化演算法的部分我並沒有選擇傳統的SGD演算法，而是選用Adam演算法(原先SGD的一種擴展版本)。

Model Training:

我們一般將做完一次批次訓練＋調整節點權數的過程叫做一輪epochs。

這裡透過設置不同的epochs，持續觀察overifitting的情況，最後得到最佳的結果，也就是落在epoch = 60附近的時候。因此我們直接將epoch設置成60。

每一次訓練都會印出accuracy來持續觀測訓練的成果，當然也有搭配loss一起看。
